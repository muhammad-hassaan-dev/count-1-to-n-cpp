# Count from 1 to N (C++)

This is a basic C++ program that prints numbers from 1 to N using a for loop.

## 💡 Concept Used:
- for loop
- user input (cin)
- output (cout)

## 📌 How it works:
1. User enters a number (n)
2. Loop runs from 1 to n
3. Each number is printed

## ▶️ Example:
Input:
5

Output:
1 2 3 4 5

## 🚀 Purpose:
This program helps beginners understand how loops work in C++.
